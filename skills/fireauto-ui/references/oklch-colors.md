# oklch() 색상 참조

## oklch(Lightness, Chroma, Hue)

- **L** (Lightness): 0 = 검정, 1 = 흰색
- **C** (Chroma): 0 = 무채색, 0.4 = 최대 채도
- **H** (Hue): 0-360도 색상환

## 색상별 참조값

| 색상 | L | C | H |
|------|---|---|---|
| 밝은 빨강 | 0.65 | 0.2 | 25 |
| 주황 | 0.75 | 0.18 | 60 |
| 황금색 | 0.8 | 0.2 | 85 |
| 초록 | 0.7 | 0.18 | 150 |
| 시안 | 0.7 | 0.15 | 190 |
| 파랑 | 0.6 | 0.2 | 250 |
| 보라 | 0.55 | 0.2 | 290 |
| 분홍 | 0.7 | 0.18 | 340 |

## 다크 테마 만들기

다크 테마에서는 base의 L 값을 낮추고, content의 L 값을 높인다:

```
base-100:     oklch(0.15 ...)   // 매우 어두운 배경
base-200:     oklch(0.12 ...)   // 더 어두운 배경
base-300:     oklch(0.1 ...)    // 가장 어두운 배경
base-content: oklch(0.9 ...)    // 밝은 텍스트
```

## 투명도 조합

```
text-base-content/60    // 60% 투명도 텍스트
bg-base-100/80          // 80% 투명도 배경
border-primary/30       // 30% 투명도 보더
```
